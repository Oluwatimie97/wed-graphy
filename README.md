# WedGraphy - Luxury Wedding Photography

A modern, responsive wedding photography booking website built with React, Vite, and Bootstrap.

## Features

- 📸 Beautiful, responsive design
- 🎨 Luxury styling with animations
- 📱 Mobile-friendly interface
- 💍 Wedding photography packages
- 🖼️ Gallery showcase
- 📝 Client reviews section
- 🛒 Booking and checkout system
- 🔍 Smooth navigation

## Technologies Used

- React 18
- Vite
- React Router DOM
- React Bootstrap
- Bootstrap Icons
- CSS3 with animations
